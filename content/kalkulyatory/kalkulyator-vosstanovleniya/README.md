# Калькулятор восстановления

Этот калькулятор оценивает **способность к восстановлению**
при текущей когнитивной нагрузке.

Он:
- не медицинский
- детерминированный
- продуктового уровня

Использует каноническую ось:
**burnout → recovery capacity → compensation**

Калькулятор опирается на каноническую ось:
**burnout → recovery capacity → compensation**.

Калькулятор предназначен для оценки риска выгорания
через изменение восстановительной способности
при текущей когнитивной нагрузке.
