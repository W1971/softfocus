# Модель интерпретации восстановления

Калькулятор описывает **recovery capacity**
как способность системы выдерживать когнитивную нагрузку
без перехода к выгоранию.

## Ключевая ось

**выгорание → восстановительная способность → компенсация**

## Логика продукта

- низкая восстановительная способность
  → высокий риск выгорания
  → требуется структурное вмешательство

- компенсация без восстановления
  → временная работоспособность
  → нарастающее истощение

## Продуктовый сигнал

При критически и нестабильно сниженной восстановительной способности
рекомендуется структурированная система восстановления,
а не точечные советы.

Калькулятор служит точкой входа
в продукт восстановления.
