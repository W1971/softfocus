# Тест на выгорание без психологии и медицинских диагнозов

Этот assessment предназначен для оценки **устойчивости когнитивной нагрузки**, а не эмоционального состояния или мотивации.

## Что именно оценивает этот assessment

Оценка основана на наблюдаемых паттернах:
- принятие решений под нагрузкой,
- устойчивость внимания,
- эффективность восстановления,
- компенсация снижения ресурса.

## Чем когнитивная перегрузка отличается от эмоционального выгорания

Когнитивная перегрузка может существовать:
- без выраженных эмоций,
- без снижения внешней продуктивности,
- без субъективного ощущения «кризиса».

## Почему субъективные ощущения ненадёжны

Большинство системных сбоев фиксируются **после** периода компенсации, а не в момент его начала.

## Наблюдаемые поведенческие сигналы

- рост усилий при сохранении результата,
- увеличение времени на решения,
- снижение постфактум-уверенности.

## Временная динамика перегрузки

Нагрузка накапливается нелинейно и проявляется с задержкой.

## Почему важно повторное измерение

Assessment предназначен для повторного использования при изменении условий нагрузки.
