# Когнитивное выгорание у разработчиков

Разработка требует длительной когнитивной непрерывности.

## Context switching

Частые переключения разрушают глубину фокуса.

## Глубокая концентрация и истощение

Глубокая работа ускоряет расход ресурса.

## Ошибки как поздний сигнал

Ошибки появляются после периода компенсации.

## Компенсация через переработки

Время используется вместо ресурса.

→ Оценить когнитивную нагрузку
