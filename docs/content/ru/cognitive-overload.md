# Умственная перегрузка: как распознать превышение когнитивной ёмкости

Умственная перегрузка возникает, когда текущие требования превышают доступную когнитивную ёмкость системы.

## Что считается когнитивной нагрузкой

Когнитивная нагрузка включает:
- количество решений,
- сложность контекста,
- частоту переключений,
- плотность ответственности.

## Признаки превышения без эмоций

Перегрузка может проявляться без стресса или тревоги.

## Почему производительность может сохраняться

Система компенсирует дефицит за счёт:
- увеличения усилий,
- удлинения рабочего времени.

## Компенсаторные паттерны

- повторные проверки,
- избыточная детализация,
- снижение скорости.

## Риски отсроченного отказа

Компенсация увеличивает вероятность резкого сбоя.
