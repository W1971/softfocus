## Fixed Template Structure

1. Front-matter
   - type: pseo
   - locale
   - path
   - status

2. H1 — Role-based framing

3. Role load context
   - structural only
   - no emotional framing

4. Why generic advice fails
   - capacity-based explanation

5. What this assessment measures
   - recovery capacity under load
   - emotional detachment
   - persistence over time

6. Non-clinical boundary

7. Assessment link
   - neutral wording only
