# Frontend Assessment Execution Contract

Status: ACTIVE

## Core UX
- Professional tone, no emotional framing
- Deterministic mapping: same answers -> same result
- Result must include (free):
  1) Burnout stage classification
  2) Primary risk vector
  3) Short interpretation
  4) Return trigger (when reassessment matters)

## Prohibited
- Gamification
- Urgency language
- Fear framing
- Medical claims

- /testy (hub)
- /test-vygoranie (core assessment entry)
- /result (shareable view)
