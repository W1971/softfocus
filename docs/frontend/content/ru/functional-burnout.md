# Функциональное выгорание: когда система работает ценой ресурса

Функциональное выгорание — это состояние, при котором результат сохраняется за счёт ускоренного расхода ресурса.

## Отличие от классического выгорания

- отсутствие эмоциональных симптомов,
- сохранение внешней эффективности.

## Почему внешне всё выглядит нормально

Компенсация маскирует системный дефицит.

## Цена поддержания уровня

Каждый цикл требует больше усилий, чем предыдущий.

## Предел компенсации

Компенсация имеет конечный предел.

## Точка системного отказа

Отказ возникает внезапно, без постепенного спада.
