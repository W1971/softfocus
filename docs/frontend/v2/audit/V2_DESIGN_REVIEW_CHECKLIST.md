# V2 Design Review Checklist

Scope: v2 only  
Mode: DESIGN_ONLY  
Audience: Design review / governance

---

## Concept Integrity

- [ ] Design does not propose features
- [ ] Design does not suggest flows
- [ ] Design does not imply user action
- [ ] Design does not promise outcomes

---

## Emotional Tone

- [ ] No motivational language
- [ ] No urgency
- [ ] No improvement framing
- [ ] No therapeutic cues

---

## Visual Behavior

- [ ] No progress indicators
- [ ] No gamification
- [ ] No completion states
- [ ] No calls to action

---

## Structural Constraints

- [ ] Static representation only
- [ ] No interaction logic
- [ ] No state transitions
- [ ] No behavioral guidance

---

## Final Gate

If any checkbox fails → **DESIGN REJECTED**

Design approval ≠ implementation approval.
