# Sleep Calculator — Product Metrics
Version: v1.0  
Status: TRACKING

---

## Purpose

Define non-marketing metrics to evaluate whether
the sleep calculator functions as a meaningful analytical tool.

---

## Core Metrics

### 1. Completion Rate

Signal:
- user reaches result screen

Target:
- ≥60% acceptable
- ≥75% healthy

---

### 2. Result Coherence

Signal:
- result perceived as logically consistent
- no surprise classification

Method:
- qualitative feedback
- internal review

---

### 3. Return Trigger Clarity

Signal:
- user understands when reassessment makes sense

Valid triggers:
- regime change
- sustained deficit
- compensation disappearance

---

### 4. Cross-Assessment Flow

Signal:
- natural transition to `/test-uroven-stressa`
- no funnel or pressure perception

---

## Non-Goals

- conversion optimization
- urgency-driven return
- gamification
