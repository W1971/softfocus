# SoftFocus — Mobile Screen Archetypes

## Archetype 1 — Assessment Screen

Purpose:
- collect responses
- avoid interpretation

Characteristics:
- simple scale
- no progress indicators
- no encouragement

---

## Archetype 2 — Result Orientation Screen

Purpose:
- describe state
- provide clarity

Characteristics:
- single dominant message
- no calls to action
- no suggestions

---

## Archetype 3 — Interpretation Reference Screen

Purpose:
- explain meaning
- reduce confusion

Characteristics:
- static text
- scroll-only
- no navigation pressure

---

## Archetype 4 — Exit Screen (Implicit)

Purpose:
- allow completion

Characteristics:
- no next step
- no prompt
- no loop

---

## Constraint

No additional archetypes may be introduced in v1.x.
