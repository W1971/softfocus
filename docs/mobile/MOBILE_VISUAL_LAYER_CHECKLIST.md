# Mobile Visual Layer Checklist (Stage 1)

This checklist governs visual presentation only.
It does not cover semantics, text meaning, or interaction logic.

## Visual Discipline

- No attention-grabbing colors
- No contrast-based emphasis
- No shadows, cards, elevation
- No gradients or decorative effects
- No success / warning / state colors

## Typography

- Neutral font weight only
- No size-based hierarchy implying importance
- No emphasis styles (bold as signal, not structure)

## Layout

- Large negative space
- Single-column layouts
- No visual grouping implying action
- No visual affordances (buttons, cards)

## Scope Limitation

- Visual tokens only
- No text meaning
- No interpretation logic

FAIL if any item is violated.
