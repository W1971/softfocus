# Assessment → Protocol Linking

## Rule

After assessment completion,
users may be offered access
to the Structured Recovery Protocol.

The protocol is optional.
No urgency or pressure is applied.

---

## Display Logic

- Assessment result is shown first
- Protocol is referenced as a supporting framework
- Language is informational, not persuasive

---

## Canonical Text (EN)

"Your result indicates reduced recovery capacity.
The Structured Recovery Protocol provides
a reference framework for understanding
and stabilizing this condition."

---

## Canonical Text (RU)

«Ваш результат указывает
на ограниченную восстановительную способность.
Структурированный протокол восстановления
предоставляет справочную структуру
для понимания и стабилизации этого состояния.»

---

## Restrictions

- No forced redirects
- No countdowns
- No bundled subscriptions
- No motivational framing
