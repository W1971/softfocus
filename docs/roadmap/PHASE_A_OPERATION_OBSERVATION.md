# Phase A — Operation & Observation

Status: ACTIVE  
Scope: System-wide  
Mutability: Locked

---

## Definition

Phase A is observation-only.

The system may:
- measure
- classify
- describe
- audit
- log

The system MUST NOT:
- prescribe actions
- optimize outcomes
- encourage behavior change
- introduce conversion mechanics
- apply urgency or motivation

---

## Allowed Outputs

- neutral states
- descriptive context
- explanatory excerpts
- reference links (non-CTA)

---

## Forbidden During Phase A

- calls to action
- recommendations
- “next steps”
- conversion copy
- reminders or nudges
- progress framing

---

## Enforcement

Any artifact violating these rules
invalidates Phase A
and MUST fail CI.
