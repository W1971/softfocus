# SoftFocus — Описание оценки

Version: v1.0  
Status: ACTIVE  
Locale: RU  
Audience: Users  

---

## Назначение

Данный документ описывает,
что представляет собой оценка в системе SoftFocus
и как она структурирована.

Описание не содержит рекомендаций
и предназначено исключительно
для понимания формата оценки.

---

## Общая характеристика оценки

Оценка в SoftFocus:

- состоит из структурированных вопросов;
- использует фиксированную логику обработки;
- не адаптируется под пользователя;
- формирует воспроизводимый результат.

Каждый запуск оценки
обрабатывается одинаково.

---

## Структура вопросов

Вопросы оценки:

- формулируются нейтрально;
- не предполагают «правильных» ответов;
- направлены на фиксацию характеристик,
  а не на интерпретацию.

Ответы используются
только в рамках формальной классификации.

---

## Результат оценки

По завершении оценки система:

- формирует результат;
- присваивает READY-классификацию;
- предоставляет формальное описание состояния.

Результат не содержит советов
и не предполагает действий.

---

## Ограничения

Оценка SoftFocus:

- не является диагностическим инструментом;
- не заменяет профессиональную оценку;
- не предназначена для принятия решений.

---

## Финальное положение

Оценка SoftFocus
предоставляет структурированные данные,
а не рекомендации или инструкции.
