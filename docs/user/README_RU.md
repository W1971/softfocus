# SoftFocus — Обзор системы

Version: v1.0  
Status: ACTIVE  
Locale: RU  
Audience: Users  

---

## Назначение

SoftFocus — это система структурированной оценки,
предназначенная для анализа состояний и паттернов
без предоставления рекомендаций или советов.

Система ориентирована на:
- фиксацию наблюдаемых характеристик;
- классификацию состояний;
- воспроизводимость результатов.

---

## Что делает система

SoftFocus:

- задаёт стандартизированные вопросы;
- фиксирует ответы в определённой структуре;
- классифицирует результат по формальным правилам;
- предоставляет интерпретацию **без рекомендаций**.

Система не принимает решений за пользователя.

---

## Что система не делает

SoftFocus:

- не даёт медицинских, психологических или иных советов;
- не предлагает действий;
- не интерпретирует результаты как диагноз;
- не заменяет профессиональную оценку.

---

## Структура взаимодействия

Работа с системой организована следующим образом:

1. Прохождение оценки
2. Формирование результата
3. Классификация состояния
4. Представление интерпретации

Каждый шаг детерминирован
и не зависит от контекста пользователя.

## Финальное положение

SoftFocus предоставляет
структурированную информацию,
а не инструкции или советы.

Ответственность за интерпретацию
и дальнейшие действия
остаётся за пользователем.
