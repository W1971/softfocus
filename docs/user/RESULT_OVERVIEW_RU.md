# SoftFocus — Обзор результата оценки

Version: v1.0  
Status: ACTIVE  
Locale: RU  
Audience: Users  

---

## Назначение

Данный документ описывает,
как представляется результат оценки в системе SoftFocus.

Описание предназначено для:
- понимания структуры результата;
- корректного чтения классификации;
- исключения неверных ожиданий.

---

## Общая структура результата

Результат оценки в SoftFocus состоит из следующих частей:

1. Сводка состояния
2. Классификация READY
3. Формальное описание выявленных характеристик
4. Уровень риска (при наличии)

Все элементы результата
формируются детерминированно.

---

## Классификация READY

Каждый результат содержит
одну из следующих классификаций:

- READY FOR DEPLOY  
- LOW RISK  
- STRUCTURAL FAIL  

Классификация отражает
**техническое состояние результата**,
а не рекомендации для пользователя.

---

## Интерпретация результата

SoftFocus предоставляет
структурированную интерпретацию,
основанную на формальных правилах.

Система:

- не даёт советов;
- не предлагает действий;
- не интерпретирует результат как диагноз.

---

## Ограничения

Результат SoftFocus:

- не является медицинским или психологическим заключением;
- не заменяет профессиональную оценку;
- предназначен исключительно для информационного анализа.

---

## Финальное положение

Результаты SoftFocus
следует рассматривать
как структурированную информацию,
а не как руководство к действию.
