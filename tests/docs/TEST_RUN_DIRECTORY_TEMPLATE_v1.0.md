# Test Run Directory Template

Каждый запуск governance создаёт **одну директорию**.

---

## Структура

tests/runs/<RUN_ID>/
├── run.log
└── run.jsonl

---

## Пример RUN_ID

RUN_2026_01_16_001

---

## Правила

- директория создаётся один раз
- файлы только дополняются
- повторное использование RUN_ID запрещено
- удаление RUN-директорий запрещено
